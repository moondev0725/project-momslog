# 👶 Momslog - 엄마들을 위한 커뮤니티 & 기록 서비스

> 육아 정보 공유, 공감 커뮤니티, 채팅 기능을 제공하는 웹 서비스

---

## 📌 프로젝트 소개

Momslog는 엄마들이 육아 정보를 공유하고,  
서로 공감하며 소통할 수 있는 커뮤니티 플랫폼입니다.

- 육아 정보 제공
- 공감 기반 게시글 시스템
- 실시간 채팅 기능
- 사용자 간 커뮤니티 형성

---

## 🎯 주요 기능

### 📝 게시글 & 커뮤니티
- 일반 게시글 작성 및 조회
- 공감(좋아요) 기능
- 댓글 작성 기능
- 공지사항 및 정보성 게시글 제공

### 💬 채팅 시스템
- 사용자 간 채팅방 생성
- 샘플 채팅 데이터 구성
- 심야 채팅 기능 구현 (Midnight Chat)

### 👩‍👧 육아 정보 제공
- 발달 정보 데이터 생성
- 평균 데이터 기반 정보 제공
- 부모 대상 정보 콘텐츠 생성

### 👤 사용자 관리
- 회원가입 / 로그인 기능
- 토큰 기반 인증 처리
- 사용자 포인트 시스템

---

## 🛠️ 기술 스택

### Backend
- Python
- Django

### Database
- SQLite3

### 기타
- REST 구조 기반 설계
- 데이터 자동 생성 스크립트 활용

---

## 📂 프로젝트 구조
accounts/ # 사용자 관리 (회원, 인증, 서비스)
posts/ # 게시글 관련 기능
chat/ # 채팅 기능
scripts/ # 데이터 생성 및 관리 스크립트

## 🚀 실행 방법

```bash
